Frequently Asked Questions

About these page-count sample PDF files.

What content is on each page?
Every page follows the same structured layout: a header bar with the file name and page number, a bold section title, three paragraphs of Lorem Ipsum body text, a sub-section with bullet points, and a footer bar with page X of Y. The first page also includes a cover title and file metadata table. All content is pure black and white — no images, no colour.
How long does the 1000-page PDF take to generate?
The 1000-page PDF generates all pages in JavaScript in your browser. This typically takes 20–40 seconds depending on your device. A progress bar tracks generation in real time. Files with 1–50 pages generate in under 2 seconds.
Are these valid, openable PDF files?
Yes. Every file is a fully valid PDF 1.7 file that opens correctly in Adobe Acrobat, Chrome, Firefox, Edge, Preview on macOS, and all other standard viewers. Page numbers, headers, and footers are consistent throughout.
Are these files safe to download?
Yes. Everything is generated locally in your browser. No data leaves your device. The PDFs contain only structured text content — no scripts, no macros, no executable code.
